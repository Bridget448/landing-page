# Landing Page with Contact Form

This is a clean and responsive landing page built to help businesses collect customer inquiries (leads) efficiently.

---

## Features

- Responsive design (works on desktop and mobile)
- Simple and modern layout
- Contact form with email functionality
- Form submissions are sent directly to email using EmailJS
- Smooth user experience with clear call-to-action

---

## Tech Stack

- HTML
- CSS
- JavaScript
- EmailJS

---

## How It Works

Users can enter their name, email, and message in the contact form.  
Once submitted, the data is sent directly to an email inbox.
